# HR Workforce Attrition Analytics Project 
## Project Overview
Analyzed HR Workforce Attrition Analytics project using Excel, SQL, and Power BI to analyze employee turnover trends, identify key attrition factors, and create interactive dashboards for data-driven workforce insights.
## Tools Used
- Excel
- SQL
- Power BI
## Key Insights
- Attrition Rate Analysis
- Department-wise Attrition
- Gender-wise Attrition
- Age Group Analysis
- Job Satisfaction Analysis
- Education-wise Attrition
## Project Structure
- SQL → All queries
- Excel → Excel file
- PowerBI → Dashboard file
- Dataset → Raw data
- Screenshots → Dashboard images
